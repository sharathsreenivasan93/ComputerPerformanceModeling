README FOR SIMULATION 1

The program is written in Python 2.7 and has been coded in Anaconda Spyder environment. To execute it on a system, Python 2.7 has to be installed.

To execute the code, open the terminal and enter the path for the python executable file (python.exe) file in the terminal (Ex. C:/Users/Sharath/Python2.7/python.exe) followed by the path of the file simulation1.py and provide the the following input for the code - 

1. Maximum value for the master clock
2. The repair time of each machine
3. Operation time of each machine
